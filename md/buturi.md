# 単振動まとめ

## 基本量

| 名称   | 記号 | 単位    | 意味       |
| ---- | -- | ----- | -------- |
| 振幅   | ==A==  | m     | 最大の揺れ幅   |
| 変位   | ==x==  | m     | 現在の位置    |
| 周期   | ==T==  | s     | 1回往復する時間 |
| 振動数  | ==f==  | Hz    | 1秒間の振動回数 |
| 角振動数 | ==ω==  | rad/s | 振動の速さ    |
| 質量   | ==m==  | kg    | おもりの重さ   |
| ばね定数 | ==K==  | N/m   | ばねの硬さ    |
